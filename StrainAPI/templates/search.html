Search Function Documentation

/search

          The search function is used to access the machine learning
          model for a prediction of the input values. The request body 
            parameters are expected to be in a json payload

Request Body Params

Name Data Type Required/optional Description
input Integer Required The input value

Request Example

              In this example the url is the web address followed
              by /search
              The json payload is {'input':10}
              the request is made to the website with the 'POST' Method

Response

response format 

                  The response will be in a json payload. The format
                  will be as follows:
Key Key Data Type Value Data Type
'id' string Integer
